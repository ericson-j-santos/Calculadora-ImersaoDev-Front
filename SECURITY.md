# Política de Segurança

## Escopo

Este repositório contém uma aplicação pública/demo para fins de estudo, portfólio e validação técnica.

## Dados sensíveis

Este projeto não deve conter:

- Tokens;
- Senhas;
- CPF, e-mail, telefone ou qualquer PII real;
- Connection strings;
- Arquivos `.env`;
- Chaves privadas;
- Dumps de banco;
- URLs internas corporativas;
- Credenciais de serviços externos.

## Reporte de vulnerabilidade

Caso encontre uma falha de segurança, abra uma issue sem expor dados sensíveis.

## Regras de publicação

Antes de publicar no GitHub Pages:

- Executar CI;
- Validar ausência de credenciais;
- Usar dados fictícios;
- Garantir que APIs reais não estejam expostas;
- Garantir que a publicação é apenas estática/demo.
